usertime.js

Output server time in usertime, it will render time in users' timezone.

Timestamp

Code

          <usertime>1441099707112</usertime>

Result

          1441099707112

ISO 8601

Code

          <usertime>2015-09-01T09:28:27+00:00</usertime>

Result

          2015-09-01T09:28:27+00:00

Code

          <usertime>2015-09-01T05:28:27-04:00</usertime>

Result

          2015-09-01T05:28:27-04:00

Time format

Code

          <usertime format="lll">1441099707112</usertime>

Result

          1441099707112
